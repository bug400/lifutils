NAME

lifilper − emulate HPIL disk drive and display for the PIL-Box

SYNOPSIS

lifilper [−n] [−s] [−d] <LIF image file> <PIL-Box tty> <scope file>

lifilper −?

DESCRIPTION

lifilper emulates a HPIL disk drive and display for the PIL-Box (J-F Garnier).

The LIF image file is only opened when a mass storage operation is performed. Otherwise the LIF image file can be modified while lifilper is running.

OPTIONS

−c

Emulate a HP 82161 cassette medium, otherwise a HP 9114A disk medium is emulated.

−d

Emulate a HP-IL display device

−s

Output HP-IL mnemonics

−?

Print a message giving the program usage to standard error.

Note: the −d and the −s option are mutually exclusive.

EXAMPLES

If disk1.lif is a LIF disk image file and the PIL-Box is connected to /dev/ttyUSB0 then

lifilper disk1.lif /dev/ttyUSB0

will emulate disk1.lif as a HP8114A disk

REFERENCES

The LIF disk directory format is documented in the HP−IL Interface Owners Manual for the HP−71 (Hewlett−Packard) with further details (particularly HP41 and HP75 file types) in the HP−41 Synthetic Quick Reference Guide (Jeremy Smith)
