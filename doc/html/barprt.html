NAME

barprt - Display an HP41 barcode file (diagnostic)

SYNOPSIS

lifutils barprt < Input file > Output file

lifutils barprt -?

DESCRIPTION

barprt is a filter which reads an HP41 barcode file from standard input. For each logical record in the barcode file, the following is printed to standard output :
The number of bytes in that row of barcode
The values of those bytes in hexadecimal
The values of those bytes in decimal

Standard output can, of course be redirected in the normal way if it is desired to pipe the output of barprt to another program or store it in a file.

barprt is intended as a diagnostic program for debugging barcode creation and printing programs. It can be used to ensure that the output of a barcode creation program is as expected. A barcode printing program can be tested by using it to print a barcode file and then comparing the output of barprt applied to that file with the result of using the HP41 command WNDSCN to read the corresponding row of printed barcode.

OPTIONS

-?

Print a message giving the program usage to standard error.

REFERENCES

See barcode(5) for details of the barcode file format

EXAMPLES

If bar1 is an HP41 barcode file then the command

lifutils barprt < bar1

will display the contents of that file in human-readable form on the user’s terminal
